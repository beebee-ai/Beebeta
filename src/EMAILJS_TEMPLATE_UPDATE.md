# EmailJS 模板更新指南

## 🎯 需要更新 EmailJS 模板以包含留言来源信息

### 📋 更新步骤

1. **登录 EmailJS 后台**
   - 访问：https://dashboard.emailjs.com/admin/templates
   - 找到您的模板：`template_rj5fksd`

2. **更新邮件内容**
   
   将模板内容修改为以下格式：

```
Subject: BEE Beta 官网留言 - {{from_name}}

Content:
您收到一条来自 BEE Beta 官网的新留言

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📍 留言来源
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
{{source}}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
👤 联系人信息
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
姓名：{{from_name}}
邮箱：{{from_email}}
电话：{{phone}}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
{{message}}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

此邮件由 BEE Beta 官网自动发送
请在24小时内回复用户
```

### 📊 模板变量说明

| 变量名 | 说明 | 示例 |
|--------|------|------|
| `{{source}}` | **新增！** 留言来源板块 | "第一阶段 - Pin 专业公开课" |
| `{{from_name}}` | 用户姓名 | "张三" |
| `{{from_email}}` | 用户邮箱 | "zhangsan@example.com" |
| `{{phone}}` | 用户电话 | "13800138000" |
| `{{message}}` | 留言内容 | "我想了解课程详情" |
| `{{to_email}}` | 收件邮箱 | "2302137484@qq.com" |

### 🎨 留言来源示例

根据用户提交留言的位置，`{{source}}` 变量会显示不同内容：

- **课程咨询弹窗**：
  - `第一阶段 - Pin 专业公开课`
  - `第二阶段A - BETA 星空营（Galaxy）`
  - `第二阶段B - BETA 深空营（Cosmos）`
  - `第三阶段 - BETA 无界营（Infinity）`

- **高级课程咨询**：
  - `Starry 繁星专业进阶课`
  - `Orbiter 伴星1v1陪练课`

- **常见问题页面**：
  - `问题常见 - 联系我们`

- **导师团队页面**：
  - `专业导师团队 - 申请邀请码`

- **联系我们页面**：
  - `联系联系我们` （来自页面标题"联系"+"联系我们"）

### ✅ 配置检查清单

- [ ] 模板已保存
- [ ] Subject 设置为：`BEE Beta 官网留言 - {{from_name}}`
- [ ] To Email 设置为：`2302137484@qq.com`
- [ ] Reply To 设置为：`{{from_email}}`
- [ ] From Name 设置为：`{{from_name}}`
- [ ] 内容中包含 `{{source}}` 变量
- [ ] 点击 "Test It" 按钮测试模板

### 📧 测试步骤

1. 在模板页面点击右上角 **"Test It"** 按钮
2. 填写测试数据：
   - from_name: `测试用户`
   - from_email: `test@example.com`
   - phone: `13800138000`
   - message: `这是一条测试消息`
   - source: `第一阶段 - Pin 专业公开课`
3. 点击 **"Send Test Email"**
4. 检查 `2302137484@qq.com` 是否收到邮件
5. 确认邮件中 **留言来源** 部分显示正确

### 🎉 完成！

更新完成后，所有从网站提交的留言都会自动包含来源信息，方便您了解用户是从哪个页面或课程提交的咨询！
